26        recipient that was refused.  Each entry contains a tuple of the SMTP
27        error code and the accompanying error message sent by the server.
28
29        This method may raise the following exceptions:
30
31         SMTPHeloError          The server didn't reply properly to
32                                the helo greeting.
33         SMTPRecipientsRefused  The server rejected ALL recipients
34                                (no mail was sent).
35         SMTPSenderRefused      The server didn't accept the from_addr.
36         SMTPDataError          The server replied with an unexpected
37                                error code (other than a refusal of
38                                a recipient).
39         SMTPNotSupportedError  The mail_options parameter includes 'SMTPUTF8'
40                                but the SMTPUTF8 extension is not supported by
41                                the server.
42
43        Note: the connection will be open even after an exception is raised.
44
45        Example:
46
47         >>> import smtplib
48         >>> s=smtplib.SMTP("localhost")
49         >>> tolist=["one@one.org","two@two.org","three@three.org","four@four.org"]
50         >>> msg = '''\\
51         ... From: Me@my.org
52         ... Subject: testin'...
53         ...
54         ... This is a test '''
55         >>> s.sendmail("me@my.org",tolist,msg)
56         { "three@three.org" : ( 550 ,"User unknown" ) }
57         >>> s.quit()
58
59        In the above example, the message was accepted for delivery to three
60        of the four addresses, and one was rejected, with the error code
61        550.  If all addresses are accepted, then the method will return an
62        empty dictionary.
63
64        """
65        self.ehlo_or_helo_if_needed()
66        esmtp_opts = []
67        if isinstance(msg, str):
68            msg = _fix_eols(msg).encode('ascii')
69        if self.does_esmtp:
70            if self.has_extn('size'):
71                esmtp_opts.append("size=%d" % len(msg))
72            for option in mail_options:
73                esmtp_opts.append(option)
74        (code, resp) = self.mail(from_addr, esmtp_opts)
75        if code != 250:
76            if code == 421:
77                self.close()
78            else:
79                self._rset()
80            raise SMTPSenderRefused(code, resp, from_addr)
81        senderrs = {}
82        if isinstance(to_addrs, str):
83            to_addrs = [to_addrs]
84        for each in to_addrs:
85            (code, resp) = self.rcpt(each, rcpt_options)
86            if (code != 250) and (code != 251):
87                senderrs[each] = (code, resp)
88            if code == 421:
89                self.close()
90                raise SMTPRecipientsRefused(senderrs)
91        if len(senderrs) == len(to_addrs):
92            # the server refused all our recipients
93            self._rset()
94            raise SMTPRecipientsRefused(senderrs)
95        (code, resp) = self.data(msg)
96        if code != 250:
97            if code == 421:
98                self.close()
99            else:
100                self._rset()
101            raise SMTPDataError(code, resp)
102        #if we got here then somebody got our mail
103        return senderrs
